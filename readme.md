# 🚀 Aplikasi Pengaduan Sarana Sekolah

---

## 📖 Deskripsi
Aplikasi Pengaduan Sarana Sekolah adalah sistem berbasis web yang dirancang untuk mempermudah siswa dalam menyampaikan laporan atau aspirasi terkait sarana dan prasarana sekolah secara digital. Melalui aplikasi ini, proses pengaduan menjadi lebih **efektif, terstruktur, dan transparan**, serta memudahkan admin dalam mengelola dan menindaklanjuti setiap laporan yang masuk.

---

## 🎯 Tujuan
* Mempermudah siswa dalam menyampaikan pengaduan sarana sekolah.
* Membantu admin dalam mengelola, memantau, dan memberikan umpan balik laporan.
* Menyediakan sistem monitoring status aspirasi secara real-time.
* Meningkatkan efisiensi penanganan masalah infrastruktur sekolah.

## 🔄 Alur Sistem

### 🧑‍🎓 Siswa
1. **Login:** Masuk ke sistem menggunakan NIS dan Password.
2. **Input:** Mengisi formulir aspirasi dan mengunggah foto bukti (Format: JPG/PNG, Max 2MB).
3. **Kirim:** Mengirim laporan ke sistem untuk diverifikasi.
4. **Monitor:** Melihat status laporan (Menunggu/Diproses/Selesai).
5. **Feedback:** Menerima umpan balik langsung dari pihak admin.

### 👨‍💼 Admin
1. **Login:** Masuk ke dashboard admin.
2. **Verifikasi:** Memeriksa laporan aspirasi yang masuk dari siswa.
3. **Proses:** Mengubah status laporan dan melakukan tindak lanjut lapangan.
4. **Feedback:** Memberikan tanggapan tertulis atas laporan yang diterima.
5. **Selesai:** Menutup laporan jika permasalahan sarana telah teratasi.

## 🛠️ Teknologi yang Digunakan
* 💻 **Bahasa Pemrograman:** PHP Native
* 🗄️ **Database:** MySQL

* ⚙️ **Environment:** Laragon

---

## 📂 Struktur Database
* `admin`: Menyimpan data kredensial petugas/admin.
* `siswa`: Menyimpan data identitas siswa.
* `aspirasi`: Menyimpan data laporan pengaduan dan statusnya.
* `kategori`: Menyimpan pengelompokan jenis sarana prasarana.

---

## ✅ Fitur Utama
* 🔐 **Login Multi User:** Akses berbeda untuk Admin dan Siswa.
* 📝 **Manajemen Aspirasi:** Fitur kirim laporan lengkap dengan unggah foto bukti.
* 📊 **Status Tracking:** Pemantauan progres penanganan laporan secara transparan.
* 💬 **Sistem Feedback:** Komunikasi dua arah antara pelapor dan petugas.
* 📚 **Histori Laporan:** Rekapitulasi pengaduan yang pernah dibuat sebelumnya.
